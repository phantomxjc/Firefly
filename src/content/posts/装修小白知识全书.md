# **第一章：装修的本质——你在买什么？**

## **1.1 装修到底在干什么？**

装修=把毛坯房变成能住的家。拆开看，它分三条线：

隐蔽工程（看不见的）：水电、防水——影响安全，出问题要砸墙

表面工程（看得见的）：墙地顶面、门窗——影响颜值

居住设备（用得着的）：定制柜、家电、软装——影响生活便利

**【重要知识点】装修的钱应该花在看不见的地方！水电防水多花1万比电视墙多花1万值100倍。因为电视墙看腻了能换，水管漏水只能砸瓷砖。**

## **1.2 装修行业的本质：信息极度不对称**

这个行业最核心的问题：业主3个月装修一次，工人天天干。

工人知道100种偷懒方法，业主一种都不知道

材料商知道100种以次充好方法，业主只知道看品牌

装修公司知道100种增项套路，业主只知道看总价

所以，你要做的就是：用知识缩小这个信息差距。

## **1.3 半包vs全包vs清包：到底怎么选？**


|  |  |  |  |
| --- | ----------- | -------- | ------------ |
| 方式 | 你做什么 | 施工方做什么 | 适合谁 |
| 清包 | 买所有材料+设计+协调 | 只出人工干活 | 有时间+懂行+有靠谱工长 |
| 半包 | 买主材+设计 | 辅材+人工+管理 | 愿意花时间学但不想太累 |
| 全包 | 出钱+验收 | 所有材料+人工 | 真没时间但有足够预算 |
|  |  |  |  |


**【避坑警告】"全包套餐999元/㎡"的真相：这只是基础价。水电点位只算15个、瓷砖只含600×600、门只含3樘、柜子只含投影5㎡——超出部分才是真正的价格。实际结算价通常是套餐价的1.8-2.5倍。**

## **1.4 装修预算到底怎么算？**

以100㎡（套内约75㎡）为例：

硬装（拆+水电+泥瓦+木工+油漆+人工）：约5-7万

主材（瓷砖/地板/门/窗/卫浴/橱柜）：约3-5万

全屋定制（衣柜+鞋柜+橱柜+浴室柜+阳台柜）：约2-4万

家电：约2-4万

软装（家具+灯具+窗帘）：约1.5-3万

合计中档：约13-23万

**【重要知识点】装修预算不是"我有20万，花完为止"。而是先确定哪些不能省（水电/防水/五金/板材环保），哪些可以以后慢慢添（装饰画/摆件/次卧家具）。把钱花在拆不掉的硬装和每天用的五金上。**

  


# **第二章：水电——家的血管系统**

## **2.1 电的基础知识**

### **你必须懂的电工常识**

电压：中国家用220V，不用管

电流：决定线要多粗。电流越大，线必须越粗

功率(W)=电压(V)×电流(A)。一个插座2500W，对应电流约11A

电线规格：1.5mm²（照明）、2.5mm²（普通插座）、4mm²（厨房/空调）、6mm²（即热热水器）

为什么？因为细线过大电流=发热=烧线=火灾。这就是为什么厨房必须用4mm²——电饭煲+微波炉+烤箱同时开会超过2.5mm²的承载能力

**【避坑警告】铝芯线便宜但导电性差+容易氧化+接头容易松动打火——很多老房子火灾就是这么来的。家用必须铜芯！看截面，铜色才是铜芯，银白色是铝芯。**

### **回路是什么？为什么重要？**

回路=从电箱出来的一条独立电路，有自己的空气开关控制。

为什么要分回路？

1. 安全：一个地方短路，只跳这一路，不会全屋黑

2. 方便维修：厨房断电不影响冰箱（所以冰箱要单独一路！）

3. 负载分散：避免一条线路承载过大电流发热

**【重要知识点】回路配置标准：照明1路、普通插座1路、厨房1路（带漏保）、卫生间1路（带漏保）、冰箱1路（单独！）、每个空调各1路。100㎡房子大约8-12路。**

### **空气开关和漏电保护器的区别**

空气开关（断路器）：电流过载/短路时跳闸 → 保护线路和设备

漏电保护器：检测到漏电（比如人触电）时跳闸 → 保护人的生命

两者都要装！尤其是厨房+卫生间+阳台等有水区域，必须带漏保

品牌：施耐德Acti9、西门子5SJ、ABB——断路器是保命的，别省

### **强弱电为什么要分开？**

强电（220V电线）会产生电磁场，干扰弱电信号（网线/电视线）

平行走线间距≥30cm，交叉处用锡纸包裹屏蔽

强弱电同管/同盒是绝对禁止的

### **什么是"死线"和"活线"？**

活线：电线穿在管里，用力能抽动 → 以后坏了可以换线不用砸墙

死线：管子拐弯太多/弯角太死，线卡住抽不动 → 坏了只能砸墙

怎么做成活线：管子拐弯用大弧弯（不用90°直角弯头），一根管不超过3根线，线径不超过管截面积的40%

## **2.2 水的知识**

### **PPR水管是什么？**

PPR=无规共聚聚丙烯，目前家用给水管的标准材料。优点：无毒、耐腐蚀、热熔连接（管子和接头融化粘成一体，不会漏水）。

冷水管：S5系列，壁厚≥2.8mm

热水管：S3.2系列，壁厚≥3.5mm（更耐热耐压）

为什么热水管更厚？热水使材料膨胀+水压冲击更大，不厚扛不住

**【重要知识点】水管品牌选择的核心不是价格，而是质保。伟星管提供"双质保卡"——免费上门打压测试+50年质保。选有厂家质保的品牌（伟星/日丰/联塑），不要选只有门店质保的杂牌。**

### **水管走顶还是走地？**

走顶：漏水马上看到（天花板湿一片），维修只需拆吊顶

走地：漏水先泡楼下才发现，维修要砸瓷砖

卫生间+厨房水管建议走顶，且必须包保温棉（防冷凝水滴落+减少热量损失）

### **什么是零冷水？为什么需要回水管？**

普通热水器：打开水龙头→放掉管子里存的冷水→才有热水。浪费水+等的烦

零冷水热水器：内置水泵循环热水，打开就是热水。但这需要回水管——让热水在管道里形成一个闭环。

没有回水管怎么办？用H阀走冷水管当回水。问题：冷水管里也有热水（冲马桶出热水），且更费燃气。

结论：水电阶段预埋回水管，不管现在用不用零冷水——反正多花几百块，以后想用随时能用。

### **存水弯是什么？为什么每个下水口都要装？**

原理：存水弯里始终存了一段水，水封隔绝管道里的臭气/蚊虫/蟑螂

不装后果：下水道的臭味和蟑螂直接进入你家

每个需要装的位置：洗手盆下水/地漏/洗衣机排水/厨房水槽

  


# **第三章：防水——家装第一生命线**

## **3.1 为什么防水是所有工序中最重要的？**

水是无孔不入的。一旦漏水：

自己家：墙皮发霉脱落、木地板鼓包、柜子腐烂

楼下邻居：天花板泡水→赔偿几千到几万+邻里关系破裂

修复代价：砸瓷砖→重做防水→重新贴砖→重新美缝，至少1万元起

所以：防水多花1000元、多刷一遍、多等2天——值。

## **3.2 防水材料：JS防水涂料的原理**

JS防水涂料=聚合物水泥防水涂料（JS是Japan Society的缩写，源自日本技术）

成分：水泥+聚合物乳液

原理：水泥提供强度和附着力，聚合物乳液填补水泥干燥后的毛细孔→形成致密的防水膜

I型（柔性）：弹性好，适合可能开裂的新旧接缝处

II型（刚性）：强度高，适合墙面（贴砖不容易掉）

家庭装修基本用II型，品牌：雨虹/德高/西卡/百得

**【避坑警告】防水涂料必须是正品！建材市场假货极多——用低价白乳胶勾兑的"防水涂料"刷上去等于没刷。去品牌专卖店或官方旗舰店买，不要在小建材店买。**

## **3.3 刷防水的正确做法**

1. 基层清理：水泥地面墙面扫干净，不能有浮灰和松动

2. 管根+墙角处理：用堵漏王（速凝型防水材料）做R角圆弧和加强层。为什么？因为直角处的防水涂料干燥时会收缩开裂，R角分散应力不开裂

3. 第一遍防水：横向涂刷

4. 第二遍防水：竖向涂刷（交叉覆盖第一遍可能漏掉的针孔）

5. 第三遍防水：再横向（三重保险）

6. 门槛石止水坎：卫生间门口用防水砂浆做一个2-3cm高的坎，防止水流到门外（这是最容易漏掉的！）

刷多高？

淋浴区墙面：到顶（热水蒸汽往上走，墙面不防水=隔壁房间墙皮返潮）

洗手区墙面：≥1.2m

卫生间其他墙面：≥0.3m

厨房/阳台地面：刷2遍，墙面刷0.3m

## **3.4 闭水试验——防水的唯一验收标准**

步骤：

1. 防水涂料完全干透后（一般24-48小时）

2. 堵住地漏，放水深度≥20mm

3. 保持至少48小时

4. 去楼下邻居家，用手电筒照天花板的每个角落+管根处

5. 无任何水渍/渗漏痕迹=合格

**【避坑警告】闭水试验不是"放了水就行了"——必须亲自去楼下看！工人/工长说的话不可全信。提前跟楼下邻居打好招呼、送点水果，这是最值得的社交投资。**

**本章核心要点：**

① 防水是装修中最不能省的环节，漏水代价极高

② JS防水涂料要买正品，去品牌店买

③ 墙角管根必须做R角处理

④ 三遍交叉涂刷

⑤ 淋浴区墙面到顶

⑥ 闭水试验48小时+亲自去楼下检查

  


# **第四章：瓷砖——你以为只是好看？**

## **4.1 瓷砖的底层知识**

### **瓷砖是怎么做出来的？**

陶土+长石+石英→压制成型→上釉→高温烧制。关键参数：

吸水率：瓷砖吸收水分的能力。越低越好（≤0.5%=全瓷砖，吸水率低=密度高=强度高=不易开裂=不渗色）

为什么吸水率重要？吸水率高的瓷砖（陶质砖/半瓷砖）会吸水膨胀→遇水容易开裂。墙面偶尔用半瓷砖（便宜），地面必须全瓷砖

通体砖vs抛釉砖：通体砖（胚体和表面同色同料，耐磨但花纹少）；抛釉砖（胚体上一层釉面，花纹丰富但釉面磨掉露底）。家用客厅推荐抛釉/柔光，性价比高

### **产地为什么重要？**

中国瓷砖产量世界第一，但质量差异大：

广东佛山：公认品质最好——高岭土矿源好+烧制工艺成熟

江西高安：中档，性价比高

山东淄博：低端为主

**⚠️ 很多品牌注册在佛山，实际生产在江西。问清楚"产地"不是"品牌注册地"。**

## **4.2 柔光砖vs亮光砖vs哑光砖**

亮光砖：光泽度高（>70°），反射强→空间显亮堂显大。问题：反光刺眼+水渍/脚印明显。适合：厨房墙面（好擦油污）

柔光砖：光泽度25-55°→最推荐！不反光刺眼+质感高级+看不见灰尘脚印。适合：客厅/卧室

哑光砖：光泽度<15°，粗犷工业风。问题：表面粗糙容易藏污纳垢不好擦。适合：阳台/卫生间地面（防滑）

## **4.3 瓷砖胶vs水泥：为什么大板上墙必须用瓷砖胶？**

传统水泥砂浆贴砖原理：水泥浆渗透进砖背面的毛细孔里→结晶固化→机械咬合

问题：现在的瓷砖越做越大/吸水性越来越低→水泥浆渗不进去→粘不住

瓷砖胶原理：水泥+高分子聚合物→既有机械咬合又有化学粘接力→粘得更牢+有一定柔性（不因楼体微震而掉砖）

C2级瓷砖胶（粘结强度≥1.0MPa）是大板和低吸水率瓷砖墙面铺贴的底线

**【避坑警告】工人说"我这水泥砂浆贴砖30年没掉过"——那是因为以前的砖小+吸水率高。你的750×1500低吸水率全瓷砖，水泥贴=半年后空鼓掉砖危险。听科学的，别听经验主义的。**

## **4.4 美缝为什么要选环氧彩砂？**

白水泥/填缝剂：最原始，便宜但会发霉变黑

真瓷胶（美缝剂）：比白水泥好但有固化收缩→缝会下凹+时间久变色脱落

环氧彩砂：环氧树脂+固化剂+彩砂填料→固化后像石头一样硬+不收缩+不吸水+不褪色。唯一缺点：贵+施工费劲

选色原则：与瓷砖同色（远看像无缝）+不要纯白（必发黄）

**本章核心要点：**

① 吸水率≤0.5%就是好砖，看产地佛山

② 客厅首选柔光砖，厨房墙面亮面砖好擦

③ 大板上墙必用瓷砖胶C2级

④ 美缝选环氧彩砂，与砖同色

  


# **第五章：地板——选错每天踩在上面难受**

## **5.1 地板种类：本质区别是什么？**

纯实木地板：一整块木头切割而成。优点：天然脚感最好。缺点：贵+会热胀冷缩+要保养+地暖不能用（会开裂变形）

三层实木复合：表层3mm+珍贵木材 → 中间层速生木材横拼→底层防潮层。优点：脚感接近实木+稳定性远优于实木（横纵交错结构抵消膨胀）。缺点：表层磨损后不可修复

多层实木复合：表层0.6-2mm薄木皮+9-11层胶合基材。优点：便宜+比三层更稳定。缺点：用胶量>三层，胶水不好的话甲醛风险

强化地板：高密度纤维板基材+印刷装饰纸+耐磨层。优点：最便宜+最耐磨+花色多。缺点：脚感硬+怕水泡+用胶量最大

SPC石塑地板：石粉+塑料。优点：100%防水+零甲醛。缺点：脚感最硬+质感塑料

## **5.2 环保等级：E0、ENF、F★★★★是什么？**

这些都是甲醛释放量标准：

E1（国标）：甲醛释放量≤0.124mg/m³ → 合格底线

E0：≤0.050mg/m³ → 更好

ENF：≤0.025mg/m³ → 国标最高级（2021年新国标）

F★★★★（日本标准）：≤0.005mg/m³ → 全球最严

结论：选E0或ENF就够，不用追F★★★★（贵且难买到正品）

**【重要知识点】"零甲醛"是营销话术。自然界中木材本身也含微量甲醛。真正该警惕的是：劣质胶水（脲醛胶）持续大量释放甲醛。好的板材用MDI胶（无甲醛添加）或酚醛胶（释放量极低）。**

**本章核心要点：**

① 首选三层实木复合，性价比+环保+脚感最优解

② 环保等级看E0/ENF，别信"零甲醛"

③ 地暖房必须地暖专用地板

④ 实木地板需要保养，懒人别选

  


# **第六章：墙面——从毛坯到光滑的秘密**

## **6.1 墙面处理的8个步骤（每一步存在的道理）**

第1步【铲墙皮】：开发商的白墙是为了通过验收刷的，用的是便宜的工程腻子（不防水不牢固）。不铲=新房墙面一年后起皮脱落。铲到水泥层为准。

第2步【刷墙锢/界面剂】：水泥墙表面有浮灰和毛细孔。墙锢渗透进水泥封闭浮灰+增加附着力。不刷=腻子粘不牢。

第3步【阴阳角找直】：用阴阳角护角条（PVC条）把墙角做直。为什么？水泥墙的墙角不可能是绝对直的。

第4步【挂网格布】：新旧墙交界+开电线槽处最容易开裂（不同材料热胀冷缩率不同）。网格布把两种材料"绑"在一起分散应力。

第5步【第一遍腻子（底层）】：粗找平，填补墙面大的不平整。厚度约1-2mm。

第6步【第二遍腻子（面层）】：精找平，面层更细腻更容易打磨。

第7步【打磨】：用砂纸把腻子表面打磨光滑。强光灯从侧面照——灯光下才能看出不平。

第8步【刷漆（一底两面）】：1遍底漆（封闭腻子碱性和增加面漆附着力）+2遍面漆（颜色和防护）。

## **6.2 为什么腻子加胶水是装修最大甲醛源之一？**

耐水腻子（美巢/德高）本身含有胶粉，加水搅拌就能用，且固化后硬度高

非耐水腻子便宜但不牢固，工人为了好刮+省力，会偷偷往非耐水腻子里加801/901胶水

801/901胶水=脲醛胶=持续释放甲醛3-15年

墙面面积是全屋最大的（约地面×3），如果全刷了含甲醛胶水的腻子=整个房间是一个巨大的甲醛释放源

对策：①合同注明禁止加胶水 ②自己买美巢YGP800耐水腻子（不需要加胶水也能施工）③腻子施工时在场盯着

## **6.3 墙面漆选购的真相**

立邦/多乐士的中低端和高端区别巨大。选"优等品"级（GB/T 9756优等品），不要"一等品"或"合格品"

环保看认证：德国蓝天使（最严格）、芬兰M1、法国A+。国产漆只要达到优等品+国标GB 18582环保标准也完全够用

乳胶漆VOC（挥发性有机物）释放高峰期在刷完后1-2周，之后快速衰减。所以油漆完成后通风2周基本就没味道了

底漆不能省！底漆的封闭作用让面漆用量省30%+颜色更均匀

**本章核心要点：**

① 开发商白墙必须铲掉

② 腻子必须耐水型，盯死不让加胶水

③ 一底两面是标配，底漆不能省

④ 油漆验收必须在强灯光下看侧面

  


# **第七章：全屋定制——最大支出、最大信息差**

## **7.1 定制柜的核心不是设计，是板材和五金**

定制柜=板材（70%）+五金（15%）+设计安装（15%）。但业主往往只关心设计，被效果图打动，忽略了板材和五金——恰好这是商家最能省钱的地方。

## **7.2 板材知识：颗粒板不是"差板"**

颗粒板（刨花板）：碎木+胶水高压压制。不是"烂木头渣"，而是木材加工剩余物利用。好不好不取决于类型，取决于：

① 用什么胶（脲醛胶=有甲醛 vs MDI胶=无甲醛添加）

② 什么工厂（大厂=德国进口压机+密度均匀+封边严实 vs 小厂=密度不均+封边开胶）

爱格板=德国爱格公司用MDI胶生产的颗粒板，全球板材天花板，F****环保

万华禾香板=国产MDI胶颗粒板，环保达到ENF，国产骄傲

露水河/大亚=国产大牌脲醛胶颗粒板，E0级，正规出厂甲醛可控但非零甲醛

多层实木板=多层薄木片交错胶合，用胶量比颗粒板多，但如果用MDI胶或好脲醛胶，也很环保

结论：板材好不好，看胶水不看来路。MDI胶板>正规大厂脲醛胶板>杂牌板

## **7.3 为什么五金比板材更影响使用寿命？**

柜子每天开合3-5次，一年1000-1800次，10年10000+次

差的铰链：用2年后门板下垂、关门哐哐响、生锈

好的铰链（百隆Blum）：带阻尼关门无声、20年都不变形

全屋约需30-60个铰链，差价10元/个×60=差600元——换来的是一整个房子的柜门品质

## **7.4 全屋定制报价的猫腻**

投影面积（最常用）：柜子宽×高，一个价格。简单但有猫腻——非标高度加钱、异形加钱

展开面积：每块板子分开算面积。精确但复尺时容易被多算

套餐陷阱套路："19999全屋定制"，实际只有20㎡投影，超出的按1200/㎡算→最终3万+

增项雷区：拉手单算吗？抽屉单算吗（标配几个？）？玻璃门单算吗？灯带单算吗？见光板（侧板颜色与门板不一致）单算吗？衣通单算吗？——全部在签合同前逐条确认

**本章核心要点：**

① 板材好坏看胶水（MDI胶>好脲醛胶>差脲醛胶）

② 五金选进口（百隆/海蒂诗），差价几百块换10年品质

③ 签合同前列增项清单逐条确认

④ 安装时现场验货，防板材被掉包

  


# **第八章：门窗——隔音保温和安全**

## **8.1 断桥铝窗户的"断桥"是什么意思？**

铝型材导热快（冬天冰冷夏天烫手），断桥=在内外铝框之间嵌入PA66尼龙隔热条→切断热传导路径→冬天不冷夏天不热

劣质断桥铝用PVC隔热条（不耐老化会收缩漏水），必须PA66尼龙

壁厚新国标≥1.8mm（2021年之前是1.4mm），越厚越结实越隔音

双层中空玻璃5+20A+5=两层5mm玻璃中间20mm空气层。空气层≥12mm才有好的隔音隔热效果，20mm是黄金值

品牌：凤铝/中财/海螺（型材大厂）、维盾/墨瑟/旭格（系统窗，贵但密封性好）

## **8.2 木门选购：别只看颜值**

实木复合门（推荐）：实木框架+实木填充+密度板贴皮。选填充率≥70%的

桥洞力学板填充：不是"纸板"，是德国技术——用刨花+胶压成管状结构，像桥梁结构一样承重+隔音。优于普通木方填充

蜂窝纸填充：最差——一戳就穿，隔音=0

品牌：TATA（静音技术好）/梦天/3D无漆木门

**本章核心要点：**

① 断桥铝要看隔热条是PA66还是PVC

② 窗户中空层≥12mm，20mm更佳

③ 木门填充物桥洞力学板>木方>蜂窝纸

  


# **第九章：装修公司的套路与应对**

## **9.1 装修公司的盈利模式**

装修公司不是靠管理费赚钱——他们靠：

① 材料差价（给你报的是零售价，他们拿货是工程价，差价30-50%）

② 增项（报价时漏项，施工中"这个要加钱那个要加钱"）

③ 管理费（合同总价10-15%）

④ 设计费（"免费设计"=设计师靠推荐材料拿回扣）

## **9.2 常见的报价漏项（签合同前对照检查）**

① 只报水电点位15个，你家实际需要50个→加钱

② 不含拆墙/铲墙皮/垃圾外运→加钱

③ 不含防水工程→加钱

④ 瓷砖铺贴不含大板/异形铺贴费→加钱

⑤ 不含美缝→加钱

⑥ 不含保洁→加钱

⑦ 不含材料搬运上楼费→加钱

⑧ 管理费/税金另算→加钱

**【避坑警告】看报价单不要只看总价！要逐项对：①这项包不包含？②包含的话数量够不够？③材料什么品牌什么型号？（必须是具体型号不是"同档次品牌"）**

## **9.3 付款方式——你的唯一筹码**

错误方式：签合同付50%，水电完付40%，竣工付10%——付了大半钱后你就没有任何谈判筹码了

正确方式：签合同付30% → 水电验收付30% → 泥木验收付30% → 竣工入住1个月后付10%

尾款要拿在手上有足够的动力让装修公司来处理问题

**本章核心要点：**

① 装修公司靠材料差价+增项赚钱，不是管理费

② 看报价逐项核对包不包含、够不够

③ 尾款至少留10%，不要提前付清

# **第十一章：甲醛——科学的认知与应对**

## **11.1 甲醛从哪里来？**

甲醛不是"装修的味道"——它是无色有刺激性气味的气体。来源按释放量排序：

1. 人造板材（颗粒板/密度板/胶合板/细木工板）——使用脲醛胶的板材——贡献最大

2. 胶粘剂（装修用胶/玻璃胶/美缝剂）

3. 油漆涂料

4. 纺织品（窗帘/地毯/沙发——为了固色防皱处理）

5. 床垫（棕垫用脲醛胶粘合=甲醛重灾区）

## **11.2 如何科学降低甲醛？**

源头控制（最重要！）：选MDI胶板材/ENF板材、选品牌墙面漆、床垫选非棕垫/知名品牌

通风（最有效！）：甲醛持续释放，通风就持续稀释。工业风扇对着窗户吹，一天换气几十次

活性炭：吸附甲醛但会饱和→需定期(1-2个月)暴晒或更换

空气净化器：选带甲醛CADR值标注的（很多净化器只能去PM2.5不能去甲醛）

绿植：心理安慰，绿萝/吊兰吸附甲醛的量杯水车薪

专业治理：光触媒/生物酶有一定效果但只能解决表面已释放的甲醛，治标不治本

**【重要知识点】甲醛释放周期3-15年，短期"除甲醛"都是暂时的。最根本的方法是：用环保材料从源头减少甲醛源+持续通风。装修完散味至少3个月再入住，有老人/小孩/孕妇的建议6个月以上。**
