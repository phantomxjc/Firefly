# 关于我 / About Me

欢迎来到我的小站。

如果你期待在这里看到一个“资深全栈工程师”或者“Certified DevOps Master”的履历，那你可能会有点失望。但如果我用一个更诚实的词来形容自己，那就是——**“会一点点的杂学家”**。

这不是谦虚，而是一种清醒的自我认知：我可能不会用 C 写操作系统，但我能用 Python 帮你把烦人的重复工作自动化；我可能没有百万级并发架构经验，但我能把家里的 NAS 和 VPS 折腾得明明白白。

以下是我的“一点点”技能树。

### 1. 代码层面：Python 是我的瑞士军刀

我不写惊天动地的算法，但我写**解决问题的脚本**。

无论是数据处理、简单的爬虫，还是给网站写个小后端，Python 是我最趁手的工具。它对我来说不是“编程语言”，而是一个**“让电脑听话的遥控器”**。虽然只是“一点点”，但这一点点足够让我在这个数字化时代过得比别人稍微舒服一点。

### 2. 硬件层面：NAS 与 VPS 的双持玩家

我对“云”这个词有执念，不过是两种截然不同的云。

- 

  **本地云（NAS）**：这是我的大后方。我享受把数据握在手里的感觉，喜欢折腾 RAID、存储池、内网穿透。对我来说，NAS 不只是硬盘柜，它是我的私人数据中心。

- 

  **公有云（VPS）**：这是我的前线阵地。我在上面学 Linux、搭网站、测服务。正如我整理的 Linux 学习笔记那样，我喜欢搞清楚 `/bin`和 `/usr/bin`的区别，喜欢用 `systemctl`管理服务的那种掌控感。**从目录结构到防火墙，我正在努力把“会用”变成“懂原理”。**

### 3. 工程层面：正在学建站的“小学生”

这也是为什么你会在这个博客上看到关于 **Twikoo 评论系统** 的折腾记录。

我正在学网站。从买域名、解析 DNS，到部署前端、对接后端。每一个步骤对我来说都是新大陆。我可能会为了一个 `duplicate`报错查半小时，也会为了把服务成功跑在 Cloudflare 上兴奋半天。**这里记录的就是我从一个“会用浏览器的人”变成一个“能让浏览器为我工作的人”的过程。**

### 4. 生活层面：吉他手

当我关掉显示器，离开机柜的嗡嗡声，我还有另一个身份——吉他手。

代码是逻辑的，吉他是感性的。左手按弦的疼痛，右手扫弦的节奏，音箱里传出的过载音色，这些东西没法用二进制解释，但它们平衡了我整天对着黑白屏幕的生活。

### 总结

这就是我。

一个 **Python 脚本写着写着就去折腾 VPS，VPS 折腾累了就去拨两下吉他** 的普通人。

这个博客不会教你什么颠覆行业的黑科技，但它会诚实地记录一个普通人如何一步步搭建自己的数字世界。

如果你也是这样的“杂学家”，欢迎留言，交个朋友。🤘
